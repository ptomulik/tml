Description

is_same checks whether two types are identical. Tha is it returns a boolean Integral Constant r such that r::value == true if and only if T1 and T2 are identical.

Expression semantics

For any types t1 and t2:

typdef is_same<t1,t2>::type r;
  • Return type: A boolean Integral Constant.
  • Semantics: r::value == true if t1 is same as t2; otherwise r::value == false.
Example
#include <tml/is_same.hpp>
using namespace tml;
struct s;
static_assert(is_same<int,int>::value == true, "");
static_assert(is_same<s,s>::value == true, "");
static_assert(is_same<s,int>::value == false, "");
