Description

variadic::c_str creates a null-terminated string from Chars....

Model of

Metafunction Class

Expression semantics

For integral Integral Constants chars... with values implicitly convertible to char

char const* r = c_str<>::apply<chars...>::value;
  • Return type: pointer to string
  • Semantics: r is a pointer to null-terminated string created from all the chars...

For integral type t and Integral Constants chars... with values implicitly convertible to t

t const* r = c_str<t>::apply<chars...>::value;
  • Return type: pointer to string
  • Semantics: r is a pointer to null-terminated string created from all the chars...
typedef c_str<>::apply<chars...>::value_type v;
typedef c_str<t>::apply<char...>::value_type v;
  • Return type: A type.
  • Semantics: v is a type representing array of characters with a size necessary to contain the string characters plus a string terminator.
Example
#include <tml/variadic/c_str.hpp>
#include <tml/char.hpp>
#include <tml/int.hpp>
#include <tml/long.hpp>
using namespace tml;
typedef variadic::c_str<>::apply<char_<'a'>, int_<'b'>, long_<'c'> > s;
constexpr char const* r = s::value;
static_assert(r[0] == 'a', "");
static_assert(r[1] == 'b', "");
static_assert(r[2] == 'c', "");
static_assert(r[3] == '\x0', "");
#include <tml/is_same.hpp>
static_assert(is_same<s::value_type, char const[4]>::value, "");
