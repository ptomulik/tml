Detailed Description

Returns the last argument from variadic argument pack.

Template Parameters
ArgsArguments to retrieve the last element from.
Synopsis
namespace variadic {
struct back
{
template <class... Args>
struct apply
{
typedef unspecified type;
};
};
}
Description

variadic::back returns the element from the end of the variadic argument pack Args... with guaranted O(1) complexity.

Model of

Metafunction Class

Expression semantics

For an any types a1, ... an

typedef variadic::back::apply<a1,...,an>::type r;
  • Return type: Unspecified type
  • Semantics: Equivalent to
    typedef an r;
Note

The maximum supported number of arguments in Args... is limited by TML_LIMIT_VARIADIC_SIZE.

Example
#include <tml/variadic/back.hpp>
#include <tml/int.hpp>
using namespace tml;
typedef variadic::back::apply<char>::type t1;
typedef variadic::back::apply<char, short, int, float>::type t4;
#include <tml/is_same.hpp>
static_assert(is_same<t1,char>::value, "");
static_assert(is_same<t4,float>::value, "");
static_assert(variadic::back::apply<char,int,int_<2> >::value == 2, "");
