Description

variadic::push_back performs an insertion at the end of the variadic argument pack Args... with guaranted O(1) complexity and applies the result to F.

Model of

Metafunction Class

Expression semantics

For an n-ary Lambda Expression f and any types a1, ... an{n-1} and any type an

typedef variadic::push_back<f,an>::apply<a1,...a{n-1}>::type r;
  • Return type: Unspecified type
  • Semantics: Equivalent to
    typedef apply<f,a1,...an>::type r;
Example
#include <tml/variadic/push_back.hpp>
#include <tml/quote.hpp>
using namespace tml;
template <class... Args> struct seq;
typedef variadic::push_back<quote<seq>, int>::apply<char, short>::type s1;
#include <tml/is_same.hpp>
static_assert(is_same< s1, seq<char, short, int> >::value, "");
