Template Sequence

Description

A Template Sequence S is an instantiation of a template class X which matches the following template parameter specification

template<class ...> struct Y

and for which the expression requirements listed below hold.

That said, any variadic and fixed-size template class with type template parameters only may fulfil the requirements of Template Sequence concept.

Refinement of

Random Access Sequence

Expression requirements

Assume that

  • S is a model of Template Sequence,
  • X is a template class matching the following template parameter specification
    template<class...> struct Y
    and such that X<T0,...T{n-1}> is identical to S for some types T0,...T{n-1},
  • i is an Integral Constant (0 <= i::value < n), and
  • F is a Lambda Expression such that
    typedef apply<F,T0,...T{n-1}>::type R;
    has constant time complexity

Then the following expressions requirements must hold

Expression Type Complexity
at< S, i > :: type Ti Constant time
front< S > :: type T0 Constant time
back< S > :: type T{n-1} Constant time
size< S > :: type Integral Constant Constant time
apply_sequence< F, S > :: type R Constant time

Expression semantics

Expression Semantics
apply_sequence< F, S > :: type Equivalent to apply< F, T1, ...Tn> :: type
