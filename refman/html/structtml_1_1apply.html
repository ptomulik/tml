Description

Invokes a Metafunction Class or a Lambda Expression with arguments Args.

Expression symantics
typedef apply<f, a1, ..., an>::type t;
  • Return type: Any type
  • Semantics: Equivalent to
    typedef apply_wrap<lambda<F>::type,a1,...,an>::type t;
Example
#include <tml/apply.hpp>
#include <tml/int.hpp>
#include <tml/placeholders.hpp>
using namespace tml;
template< class N1, class N2 >
struct int_plus
: int_<( N1::value + N2::value )>
{ };
typedef apply< int_plus<_1,_2>, int_<2>, int_<3> >::type r;
static_assert(r::value == 5, "");
