Description

quote is a higher-order primitive that wraps a Metafunction to create a corresponding Metafunction Class.

Model of

Metafunction Class

Expression semantics

For any Metafunction f and arbitrary type tag:

typedef quote<f> g;
typedef quote<f,tag> g;
  • Return type: Metafunction Class

  • Semantics: Equivalent to:

    struct g
    {
    template< class... Args > struct apply
    : f<Args...>
    {
    };
    };

    if f<Args...> has nested type member type, and to

    struct g
    {
    template< class... Args > struct apply
    {
    typedef f<Args...> type;
    };
    };

    otherwise.

Example
#include <tml/quote.hpp>
using namespace tml;
template <class T>
struct f1
{
typedef T type;
};
template<class T1, class T2, class T3, class T4, class T5>
struct f5
{
// no 'type' member!
};
typedef quote<f1>::apply<int>::type t1;
typedef quote<f5>::apply<char,short,int,long,float>::type t5;
#include <tml/is_same.hpp>
static_assert(is_same<t1, int>::value, "");
static_assert(is_same<t5, f5<char,short,int,long,float> >::value, "");
