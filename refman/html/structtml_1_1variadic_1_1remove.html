Description

remove performs a removal of a single argument from the argument pack at the position N with guaranteed O(1) complexity.

Model of

Metafunction Class

Expression semantics

For an n-ary Lambda Expression f, an Integral Constant i, and any types a0,...,ai,...an

typedef variadic::remove<f,i>::apply<a0,...,ai,...an>::type r;
  • Return type: Unspecified type
  • Semantics: equivalent to
    typedef apply<f,a0,...,a{i-1},a{i+1},...a{n+1}>::type r;
Example
#include <tml/variadic/remove.hpp>
#include <tml/quote.hpp>
#include <tml/int.hpp>
using namespace tml;
template <class... Args> struct seq;
typedef variadic::remove<quote<seq>, int_<1> >::apply<char, short, int, float>::type s1;
typedef variadic::remove_c<quote<seq>, 2>::apply<char, short, int, float>::type s2;
#include <tml/is_same.hpp>
static_assert(is_same< s1, seq<char, int, float> >::value, "");
static_assert(is_same< s2, seq<char, short, float> >::value, "");
