Description

Returns a tag identifying given sequence. The returned tag is used by Tag Dispatched Metafunctions operating on sequences to dispatch operation to an appropriate implementation.

Model of

Metafunction

Expression semantics

For any type s

typedef sequence_tag<s>::type t;
  • Return type: Unspecified
  • Semantics: By default equivalent to
    typedef s::sequence_tag t;
    t is a type used to dispatch execution of Tag Dispatched Metafunctions operating on sequences to appropriate implementations.
