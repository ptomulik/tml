Description

c_str converts a sequence containing characters into a null-terminated string.

Model of

Metafunction

Expression semantics

For a sequence s of Int Constants with values implicitly convertible to char

char const* r = c_str<s>::value;
  • Return type: pointer to string
  • Semantics: r is a pointer to null-terminated string created from all the characters contained in sequence s.

For an integral type t and a sequence s of Int Constants with values implicitly convertible to t

t const* r = c_str<s,t>::value;
  • Return type: pointer to string
  • Semantics: r is a pointer to null-terminated string created from all the characters contained in sequence s.
typedef c_str<s>::value_type v;
typedef c_str<s,t>::value_type v;
  • Return type: A type.
  • Semantics: v is a type representing array of characters with a size necessary to contain the string characters plus a string terminator.
typedef c_str<s>::type f;
  • Return type: A type.
  • Semantics: Equivalent to
    typedef c_str<s> f;
Example
#include <tml/c_str.hpp>
#include <tml/sequence.hpp>
using namespace tml;
typedef c_str<sequence_c<char,'a','b','c'> >::type s;
constexpr char const* r = s::value;
static_assert(r[0] == 'a', "");
static_assert(r[1] == 'b', "");
static_assert(r[2] == 'c', "");
static_assert(r[3] == '\x0', "");
#include <tml/is_same.hpp>
static_assert(is_same<s::value_type, char const[4]>::value, "");
static_assert(is_same<s::type, s>::value, "");
